# Nivela 📊

O **Nivela** é uma plataforma robusta desenvolvida sob demanda corporativa para a padronização, auditoria e nivelamento de processos internos, garantindo que métricas de desempenho e conformidade sejam seguidas com precisão.

> **Nota de Compliance:** Este repositório funciona como um *Estudo de Caso* para fins de portfólio. O código-fonte original e as regras de negócio específicas são privados e protegidos por acordos de confidencialidade (NDA).

---

## 🎯 O Desafio
A organização precisava de uma ferramenta confiável para auditar o progresso de equipes e nivelar a qualidade das entregas com base em critérios rigorosos de avaliação, gerando históricos consultáveis para tomadas de decisão estratégicas.

## 🚀 Minha Contribuição e Solução
Fui responsável pelo desenvolvimento do ecossistema do aplicativo em Java, focando em manutenibilidade e experiência do usuário:

* **Modelagem de Dados Precisa:** Estruturação de entidades complexas e relacionamentos em SQL para garantir a integridade dos dados históricos de avaliação.
* **Interface Gráfica sob Medida:** Construção manual de telas interativas utilizando Java Swing. O foco foi criar um layout limpo, focado em produtividade e sem a poluição gerada por ferramentas de "arrastar e soltar".
* **Segurança e Validação:** Implementação de rotinas rígidas de validação de dados nas telas, impedindo a inserção de informações inconsistentes no banco de dados.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java
* **Interface Gráfica:** Java Swing (Interface Limpa e Manual)
* **Padrões de Projeto:** DAO (Data Access Object) para isolamento de consultas SQL
* **Ambiente de Desenvolvimento:** Apache NetBeans

---

## 📈 Resultados Obtidos
* Centralização e segurança no armazenamento de métricas de conformidade.
